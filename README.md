# Donor-profiling-agent

Developed an AI-driven donor intelligence system to generate structured account insights
Designed JSON-based summaries covering donor behavior, engagement metrics, and opportunity pipelines
Automated analysis of CRM data including activities and giving history
Delivered actionable insights directly within Salesforce

Impact:
Improved donor engagement strategies
Enabled data-driven decision-making
Consolidated complex data into clear insights
